# Wordly-C++

## Overview

 *This project is clone of the `wordly` online game, built with C++, and raylib for graphic environment*. Word is chosen randomly from provided dictionary.

 ## Getting started

 ```bash
git clone https://github.com/quaslir/Wordly-C-/
cd Wordly-C
cd build
cmake ..
make
./Wordly-C++ `your_dictionary`
```
